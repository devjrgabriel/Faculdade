Se o usuário escolher a opção 1, exiba "Opção 1 escolhida". Se o usuário escolher a opção 2, exiba "Opção 2 escolhida". Se o usuário escolher a opção 3, exiba "Opção 3 escolhida". Se o usuário escolher uma opção inválida, exiba "Opção inválida".
